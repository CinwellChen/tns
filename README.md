# thriftnameserver

thriftnameserver是配合thrift使用的工具，为thrift提供分布式方案，通过该工具实现thrift服务的发现、高可靠、负载均衡、水平扩展

详细用法请参考[wiki](https://git.ws.700bike.com.cn/pubservice/thriftnameserver/wikis/home "thriftnameserver wiki")
